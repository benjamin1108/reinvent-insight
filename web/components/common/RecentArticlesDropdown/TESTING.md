# RecentArticlesDropdown 测试指南

## 功能测试清单

### 桌面端测试

#### 基础功能
- [ ] 鼠标悬停在"近期解读"按钮上，下拉列表显示
- [ ] 鼠标移开按钮，300ms后下拉列表关闭
- [ ] 鼠标移入下拉列表，列表保持显示
- [ ] 鼠标移出下拉列表，300ms后列表关闭

#### 数据加载
- [ ] 首次悬停时自动加载文章数据
- [ ] 显示加载动画
- [ ] 数据加载成功后显示文章列表
- [ ] 数据按修改时间倒序排列
- [ ] 最多显示10篇文章

#### 文章显示
- [ ] 文章标题正确显示（优先中文标题）
- [ ] 相对时间正确显示（如"2小时前"）
- [ ] 内容类型标签正确显示
- [ ] 文章标题过长时正确截断（最多2行）

#### 交互功能
- [ ] 点击文章条目，正确导航到文章页面
- [ ] 点击后下拉列表立即关闭
- [ ] 文章条目悬停时显示高亮效果
- [ ] URL更新为 /d/{hash} 格式

#### 空状态
- [ ] 无文章时显示"暂无近期解读"提示
- [ ] 空状态图标和文字正确显示

#### 错误处理
- [ ] API请求失败时不阻塞界面
- [ ] 无效数据被正确过滤
- [ ] 时间戳异常时显示"未知时间"

### 移动端测试

#### 基础功能
- [ ] 点击"近期解读"按钮，下拉列表显示/隐藏切换
- [ ] 点击页面其他区域，下拉列表关闭
- [ ] 下拉列表宽度适应屏幕

#### 触摸交互
- [ ] 文章条目触摸目标足够大（最小44px）
- [ ] 触摸文章条目，正确导航
- [ ] 滚动流畅，无卡顿

#### 响应式布局
- [ ] 在不同屏幕尺寸下布局正确
- [ ] 极小屏幕（<480px）下可用性良好
- [ ] 横竖屏切换正常

#### 键盘导航

- [ ] ARIA属性正确设置

- [ ] 下拉列表角色正确标识

### 性能测试

#### 缓存机制

#### 动画性能
- [ ] 淡入动画流畅
- [ ] 悬停效果无延迟
- [ ] 滚动性能良好

### 浏览器兼容性

## 测试步骤

### 1. 启动应用
```bash
# 启动开发服务器
python -m reinvent_insight.api
```

### 2. 登录系统
- 访问 http://localhost:8001
- 使用管理员账号登录

### 3. 测试桌面端
- 将浏览器窗口调整到桌面尺寸（>768px）
- 按照桌面端测试清单逐项测试

### 4. 测试移动端
- 打开浏览器开发者工具
- 切换到移动设备模拟模式
- 按照移动端测试清单逐项测试

### 5. 测试可访问性
- 使用Tab键导航
- 使用屏幕阅读器测试
- 检查ARIA属性
