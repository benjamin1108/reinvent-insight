# 快速开始 - 多格式文档支持

## 🚀 5分钟快速上手

### 1. 启动服务

```bash
# 开发环境
./run-dev.sh

# 或生产环境
reinvent-insight web
```

### 2. 访问 Web 界面

- 开发环境：http://localhost:8002
- 生产环境：http://localhost:8001

### 3. 登录系统

使用配置的用户名和密码登录

### 4. 上传文档

1. 点击"文档文件"标签页
2. 选择或拖拽文档文件
3. 支持的格式：
   - ✅ TXT（文本文件，最大 10MB）
   - ✅ MD（Markdown，最大 10MB）
   - ✅ PDF（PDF 文档，最大 50MB）
   - ✅ DOCX（Word 文档，最大 50MB）

### 5. 开始分析

点击"分析文档文件"按钮，系统会：
- 上传文件到服务器
- 实时显示上传进度
- 自动开始分析
- 显示分析进度
- 生成深度分析报告

### 6. 查看报告

分析完成后，点击"点击阅读"查看完整报告

## 🔧 配置（可选）

在 `.env` 文件中可以配置：

```bash
# 文本文件最大大小（字节）
MAX_TEXT_FILE_SIZE=10485760  # 10MB

# 二进制文件最大大小（字节）
MAX_BINARY_FILE_SIZE=52428800  # 50MB
```

## 🐛 常见问题

### Q: 上传失败，提示"不支持的文件格式"
**A**: 请确保文件扩展名为 .txt, .md, .pdf 或 .docx

### Q: 提示"文件大小超过限制"
**A**: 
- 文本文件（TXT/MD）：最大 10MB
- 二进制文件（PDF/DOCX）：最大 50MB

### Q: PDF 分析结果不准确
**A**: 请确保 PDF 是文本格式而非扫描版

### Q: 文本文件编码错误
**A**: 建议使用 UTF-8 编码保存文件
